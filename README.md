# A2Banking
La entidad financiera A2 Banking como parte de su proceso de modernización tecnológica, ha 
decidido desarrollar un sistema especializado para la gestión de cuentas bancarias que 
permita realizar operaciones de depósito, retiro y consulta de saldo de manera segura, 
trazable y escalable. 

El software deberá implementarse bajo la Arquitectura Cliente-Servidor y Arquitectura en N 
Capas, integrando los patrones de diseño, y utilizando procedimientos almacenados con 
transacciones en la base de datos, consumo de servicios externos a través de API, calculo de 
comisiones mediante SQL CLR, y autenticación biométrica mediante huella digital.
